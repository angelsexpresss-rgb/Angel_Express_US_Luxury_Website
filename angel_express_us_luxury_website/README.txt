- Online booking form
- WhatsApp message buttons
- Google Calendar button
- Working local backend
- Customer database and booking records
- CSV export

PREVIEW WITHOUT BACKEND:
Open public/index.html in Google Chrome.

RUN WITH WORKING BACKEND:
1. Install Node.js
2. Unzip this folder
3. Open terminal in this folder
4. Run: npm install
5. Run: npm start
6. Open: http://localhost:3000

CHANGE WHATSAPP NUMBER:
Open public/script.js and replace:
OWNER_WHATSAPP = "12145550199"
with your number using country code only.

NOTE:
The website opens prepared WhatsApp messages. True automatic WhatsApp sending requires WhatsApp Business Cloud API or Twilio.
